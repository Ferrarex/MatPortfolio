MATEMÁTICA FINANCEIRA

A matemática financeira se divide em 2 partes:

  • Sistema de Capitalização

  • Sistema de Amortização


Sistema de Capitalização
Fórmula: M = P [ (1 + i)n / i ]

    Serve para calcular um valor futuro de vários depósitos iguais. Um exemplo seria uma conta poupança, onde é depositado um valor todo mês, nessa situação, "P" seria o valor depositado, "N" seriam os meses, M seria o muntante e "i" seria o juro. (Analizado a partir de uma das questões de exemplo).

    Para fazer calculos de capitalização normalmente é utilizado uma calculadora, pois se perde precisão após arredondar várias vezes.


Sistema de Amortização
Fórmulas: C = P [(1 + i)n - 1 / i (1 + i)n] ou P = C [i (1 + i)n / (1 + i)n - 1 ]

    Amortização é o contrário, é fazer pagamentos para acabar com uma dívida existente. Existem duas fórmulas, dependendo do valor que é preciso encontrar (Valor das Prestações ou o Capital Inicial).

    Nesse se mostra muito mais importante o uso da calculadora para resultados precisos.

Geral

  1. Se o tamanho de linhas é diferente do tamanho de colunas, a matriz é retangular, casó contrário, quadrada
  2. Ao representar elementos, se segue o padrão aij ("i" sendo linha e "j" sendo coluna)
  3. Uma matriz de só uma coluna é chamada "matriz coluna", e uma matriz de só uma linha é chamada "matriz linha"
  4. Quando todos os termos de uma matriz são 0, ela é uma matriz zerada
  5. Uma matriz é diagonal quando o valor de aij é zero em todas as casas que não sejam da diagonal principal
  6. Matriz identidade é uma matriz diagonal só com 1
  7. Matriz triangular superior ou inferior acontece quando os termos acima ou abaixo da diagonal principal são zerados

Determinantes

    Determinante é o número que uma matriz quadrata tem. pode ser representada por det(M), |M| ou det(Ml x c) (retirado dos slides). Cada um dos tamanhos tem formas diferentes de se calcular o determinante.

    Não entendi o uso dos determinantes em situações do dia a dia.

Propriedades gerais de determinantes:

  1. Uma matriz com uma linha ou coluna zerada, terá seu determinante 0
  2. Quando há duas filas paralelas iguais ou proporcionais, seu determinante será 0
  3. A determinante de uma matriz e a determinante da mesma matriz composta é igual
  4. Caso multiplique uma fila por um número, o determinante também será multiplicado pelo mesmo
  5. Ao trocar filas, o determinante é multiplicado por -1.
  6. Binet: se A e B são matrizes de mesmo tamanho, o determinante de (A*B) é igual o determinante de A multiplicado pelo determinante de B
  7. O determinante de uma matriz triangular é o produto dos elementos da diagonal principal. (pego dos slides)
  8. Jacob: Não entendi o Teorema

Determinantes de tamanho 1 ( M1 x 1 )

    A mais fácil de todas, o 11 da matriz já é o próprio determinante.

M =

M11

det(M1x1) = M11


Determinantes de tamanho 2 ( M2 x 2 )

    Para encontrar o determinante em matrizes 2x2, é necessário fazer a multiplicação dos termos da diagonal principal e subtrair com o produto dos termos da diagonal secundária.

M =

M11M12
M21M22

det(M2x2) = M11 * M22 - M12 * M21


Determinantes de tamanho 3 ( M3 x 3 )

    Para encontrar o determinante em matrizes 3x3, é necessário repetir as primeiras 2 colunas, e fazer 3 diagonais principais e 3 diagonais secundárias. Após isso, basta repetir o processo do determinante 2x2.

M =

M11M12M13
M21M22M23
M31M32M33

det(M3x3) = (M11 * M22 * M33 + M12 * M23 * M31 + M13 * M21 * M32) - (M13 * M22 * M31 + M11 * M23 * M32 + M12 * M21 * M33)
